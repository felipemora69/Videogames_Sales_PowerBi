# Video Game Sales Dashboard – Power BI

**- Project Overview**
This project analyzes a historical video game sales dataset. The main goal is to visualize total sales by year and region, while allowing users to explore the data through dynamic filters by genre, platform, publisher, and year.

**Tables Used:**
- Sales Table – Contains sales by region, platform, genre, publisher, and release year
- Power Query Transformations:
- Cleaned and standardized column names
- Translated country and region names into Enflish
- Calculated Field:
- A simple measure was created to summarize total global sales
Note: No separate metrics table or complex DAX model was used in this version.

**Dashboard Features**
- KPI Card: Displays total global sales
- Bar Chart: Shows sales trends by year and region
- Pie Chart: Visualizes profit distribution by continent
- Slicers:
- Year
- Region
- Genre
- Platform
- Publisher
- Detailed Table: Displays game-level data with all relevant fields in English
- 
**Tools Used**
- Power BI Desktop
- Power Query for basic ETL and column transformations
- Excel (original data source)
