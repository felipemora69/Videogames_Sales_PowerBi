**- Project Overview**
This project analyzes a historical video game sales dataset. The main goal is to visualize total sales by year and region, while allowing users to explore the data through dynamic filters by genre, platform, publisher, and year.

**- Analytical Objectives**
- Display total sales by year and region
- Enable interactive filtering by:
- Year
- Region
- Genre
- Platform
- Editorial
- Present a detailed table with game-level information:
- Title
- Platform
- Genre
- Publisher
- Release Year
- Regional sales (NA, EU, JP, Others)
- Global sales
  
**Tools Used**
- Power BI Desktop
- Visuals: bar charts, slicers, dynamic table
- Full translation of fields and labels into Spanish
- Excel (original data source)
- Power Query transformations for cleaning and standardization

** Dashboard Features**
- Clean, intuitive layout
- Interactive filters for exploratory analysis
- Final table with all relevant fields in Spanish
- Useful for market research, trend analysis, and editorial decision-making
